@webboot/magic-theme

this is the @magic-themes webboot theme. It is being used as the foundational theme for all @webboot pages.

installation

npm install --save --save-exact @webboot/magic-theme

usage

require in config.mjs:

// config.mjsexport default {  // ... other config  THEME: '@webboot/magic-theme',}

magic will then find and merge the themes automatically, just as it did on this page
