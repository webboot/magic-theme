@webboot/magic-theme

this is the @magic-themes webboot theme. It is being used as the foundational theme for all @webboot pages.

currently, this theme is being used on the following pages:

  • webboot.org
  • explain.webboot.org

installation

npm install --save --save-exact @webboot/magic-theme

usually, webboot pages also use the project theme

usage

require

in /config.mjs:

// /config.mjsexport default {  // ... other config  THEME: ['project', '@webboot/magic-theme'],}

theme vars

colors

this theme exports and uses the following colors.

  • vars.white: #ffffff
  • vars.neutral: #5a5a5a
  • background
    • vars.background.light: #e0e0e0
    • vars.background.dark: #212121
  • link
    • vars.link.light: #212121
    • vars.link.dark: #c4c4c4
    • link.hover
      • vars.link.hover.light: #ea4444
      • vars.link.hover.dark: #ea4444
  • primary
    • vars.primary.neutral: #ea4444
    • vars.primary.light: #ff7070
    • vars.primary.dark: #c82222
  • secondary
    • vars.secondary.neutral: #44ea44
    • vars.secondary.light: #70ff70
    • vars.secondary.dark: #22c822
  • text
    • vars.text.light: #212121
    • vars.text.dark: #c4c4c4

max page width

vars.maxWidth: 1000px

fade duration:

vars.fadeDuration should be used for all css animations

vars.fadeDuration: 500ms

breakpoints

the following width breakpoints exist

  • vars.widths.tablet: 500px
  • vars.widths.laptop: 1024px
  • vars.widths.desktop: 1600px
