@webboot/magic-theme

this is the @magic-themes webboot theme. It is being used as the foundational theme for all @webboot pages.

installation

npm install --save --save-exact @webboot/magic-theme

usage

require in config.mjs:

// config.mjsexport default {  // ... other config  THEME: '@webboot/magic-theme',}

colors

@webboot/magic-theme exports and uses the following default colors:

  • grey : #5a5a5a
  • darkText : #c4c4c4
  • backgroundDark : #212121
  • backgroundLight : #e0e0e0
  • lightText : #212121
  • red : #ea4444
  • redLight : #ff7070
  • redDark : #c42222
  • white : #ffffff
  • linkColor : #c4c4c4
  • linkColorLight : #212121
  • linkColorHover : #ea4444
  • linkColorHoverLight : #ea4444

max page width

1000px

fade duration:

vars.fadeDuration should be used for all css animations

fadeDuration: 500ms

widths

the following width breakpoints exist:

  • tablet: 500px
  • laptop: 1024px
  • desktop: 1600px
